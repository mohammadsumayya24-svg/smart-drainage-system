# Smart Drainage Waste Collection System

## Description
This project is a computer vision-based system that detects waste in drainage using a camera. It simulates an automated waste collection process and monitors bin levels.

## Features
- Detects waste using camera
- Simulates waste collection
- Displays bin level
- Shows warning at 80% capacity
- Alerts when bin is full

## Technologies Used
- Python
- OpenCV
- NumPy

## How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the project:
   python main.py

## Output
- Real-time waste detection
- Bin level monitoring
- Alert when bin is full
